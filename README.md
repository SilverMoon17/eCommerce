# eCommerce Pet-Project

Welcome to our eCommerce pet-project! This project is aimed at building a fullstack application for managing an online store, focusing on pet products. It's built using Docker, T-SQL, React TypeScript, and .NET WEB API.

## Features

- **Product Management:** Easily add, edit, and remove products from the store.
- **User Authentication:** Secure user registration and authentication system.
- **Shopping Cart:** Users can add products to their cart and proceed to checkout.
- **Order Management:** Admins can manage orders and track shipment status.
- **Responsive Design:** The UI is designed to work seamlessly across devices.

## Technologies Used

- **Docker:** Containerization platform for packaging applications and their dependencies into standardized units for deployment.
- **T-SQL:** For database management and data manipulation.
- **React TypeScript:** Frontend framework for building interactive user interfaces.
- **.NET WEB API:** Backend framework for building RESTful APIs and handling business logic.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/SilverMoon17/eCommerce.git
```
### For frontend

```
cd frontend
npm install
npm start
```

### For backend

```
cd backend
dotnet run
```

Access the application:

Open your web browser and navigate to http://localhost:3000 to view the application.
